# datavjs is a JavaScript library for data visualization.

## Requires:
 * [d3.js](https://github.com/mbostock/d3).
 * [raphael.js](http://raphaeljs.com/).
 * [sea.js](https://github.com/seajs/seajs).

## Want to learn more?
   The example site: <http://datavlab.org/datav.js/>
